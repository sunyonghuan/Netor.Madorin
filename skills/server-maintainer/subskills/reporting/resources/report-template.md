## 5. 建议处理事项
1. **[192.168.1.20]**: 执行 `free -m` 和 `top` 确认占用内存的进程，考虑重启服务或增加 Swap。
2. **[192.168.1.30]**: 清理 `/var/log` 下的大文件，检查 `du -sh *` 定位大目录。
3. **[192.168.1.10]**: 查看 `/var/log/auth.log`，封禁恶意 IP。
4. **[全局]**: 建议检查所有服务器的内核更新情况，部分服务器运行时间超过 30 天。

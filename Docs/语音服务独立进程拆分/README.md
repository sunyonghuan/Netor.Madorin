# 语音服务独立进程拆分方案

> 创建日期：2026-04-20
> 状态：方案设计完成，等待审核后逐步执行

## 背景

当前语音模块（KWS/STT/TTS）作为类库直接集成在主进程中，耦合度高，
更新语音模型需要重新编译整个应用。拆分为独立进程后，用户可以独立更新
STT 和 TTS 服务，主程序无需变动。

## 核心决策

| 组件 | 决策 | 理由 |
|---|---|---|
| KWS 唤醒词 | **留在主进程** | 体积小、替换概率低、无独立更新需求 |
| STT 语音识别 | **独立进程** `Cortana.Stt.exe` | 不同模型各有所长，需独立选择/更新 |
| TTS 语音合成 | **独立进程** `Cortana.Tts.exe` | 不同模型各有所长，需独立选择/更新 |

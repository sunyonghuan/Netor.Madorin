# Netor.Madorin v1.1.6 发布说明

**发布日期：** 2026 年 4 月 18 日
**运行时：** .NET 10 | Avalonia 12 | Native AOT
**平台：** Windows 10/11 x64

---

## 版本亮点

本版本聚焦三件事：聊天界面可读性继续优化、PowerShell 工具链稳定性修复、MCP 离线连接噪音控制。相比 v1.1.5，这一版更适合作为日常主力版本使用。

---

## 新增与改进

### 聊天气泡视觉细节优化

- 调整头像尺寸、气泡圆角和左右留白，消息区整体更均衡。
- 改善连续对话时的视觉层级，减少拥挤感。
- 适配当前 UI 主界面布局。

### PowerShell 回归验证补充

- 新增最小回归测试程序，覆盖 PowerShell 执行器和持久会话的关键稳定性场景。
- 验证“调用方 CancellationToken 已取消”与“命令执行超时”两类路径。

## 问题修复

### PowerShell 工具执行过早中断

- **现象：** 在 AI 触发 PowerShell 脚本或会话命令时，进程可能在几十毫秒内被取消，导致最后输出拿不到。
- **原因：** 进程执行生命周期直接绑定了 AI 框架传入的 CancellationToken，上游取消会误杀实际执行中的 PowerShell 进程。
- **修复：** 将执行锁、命令超时和进程等待逻辑与外部 CancellationToken 解耦，仅由显式 timeout 控制执行生命周期。
- **效果：** 避免脚本刚启动就被取消，超时行为仍然保留。

### MCP 服务离线时持续重连刷日志

- **现象：** 当 MCP 对端下线、地址不可达或连接失败时，宿主端可能持续尝试连接，并在日志中反复输出错误。
- **原因：** 连接失败后的 McpClient/transport 资源没有被完整释放，后台传输层继续重试。
- **修复：** 为 MCP 连接增加 15 秒超时，并在连接失败或超时时主动清理 client 与未接管的 transport；补齐调用侧失败时的 Dispose。
- **效果：** 对端不可达时不再无限重连，也不会持续刷错误日志。

### Markdown HTML Inline 显示异常

- **现象：** 聊天气泡中可能显示 HtmlInline 类型名而不是预期内容。
- **修复：** 正确处理 `<br>` 换行，并忽略其他不应直出的 HTML inline。

### 最后一行文本渲染异常

- **现象：** 某些聊天气泡的最后一行文本显示异常或被裁切。
- **修复：** 移除不合适的固定行高设置，恢复 Avalonia 默认排版行为。

## 工程与安全改进

- 补充本地运行态数据库文件忽略规则，避免把 SQLite 的 `-wal` / `-shm` 临时文件带入版本库。
- 补充本地服务账号密钥文件忽略规则，降低误提交敏感凭据的风险。
- 继续完善插件与测试基础设施，为后续工具扩展和回归验证做准备。

---

## 升级说明

1. 从 v1.1.5 升级可直接替换程序文件。
2. 本版本不包含破坏性配置变更。
3. 如果你使用 MCP 远程工具，升级后建议重新验证离线、超时和异常连接场景的日志表现。

## 已知事项

- 仓库中的旧 WinForms 项目 `Src/Netor.Madorin` 仍存在历史依赖版本冲突，不影响当前 UI 主线发布。
- Avalonia 编译仍会提示少量已知过时 API 警告，不影响当前版本运行与发布。

# 01 - 布局规格

> 本文档定义新版界面的控件树、尺寸、对齐、联动逻辑、设计 token 表。
> 上级文档：[README.md](./README.md)

---

## 1. 整体框架

### 1.1 窗口规格

| 维度 | 值 | 说明 |
|---|---|---|
| 默认尺寸 | 560 × 980 | 决策 UI-10：窄竖屏，单手桌面右侧驻留场景 |
| 最小尺寸 | 480 × 640 | 低于此尺寸不可拖小 |
| 最大尺寸 | 不限 | 用户可手动拖大，主区按比例扩展 |
| 启动位置 | CenterScreen | 与现有保持一致 |
| 标题栏 | ExtendClientAreaToDecorationsHint=True | 自定义 36px 顶栏 |

### 1.2 三段结构（顶栏 + 左侧 + 主区）

```
Window (560×980)
├─ TitleBar (Height=36)        ← 顶栏：模式 tab + 全局操作
└─ Main Grid (Row=1, *)
   ├─ LeftPanel (Width=240, MinWidth=180, MaxWidth=360)
   ├─ GridSplitter (Width=4)
   └─ MainContent (Width=*)
```

**左侧面板宽度策略**：
- 默认 240px
- 可拖动调整（GridSplitter）
- 双击拖条恢复 240px
- 拖至 < 80px 时隐藏左侧（用户用顶栏 ☰ 重新打开）

## 2. 顶栏（Height = 36px）

### 2.1 顶栏控件树

```
TitleBar (36×*)
├─ ☰ 抽屉按钮 (Left)          ← 折叠/展开左侧面板
├─ Logo + 标题 "小娜宝贝"      ← 仅 >= 720px 宽度时显示
├─ 模式 Tab 组 (Center)
│  ├─ Tab "对话" (active 高亮)
│  ├─ Tab "工作流"
│  └─ Tab "群聊"
├─ 最近 ▼ 下拉按钮 (Right)
├─ ⚙ 设置按钮
└─ 系统按钮 (— □ ✕)
```

### 2.2 顶栏元素尺寸

| 元素 | 尺寸 | 间距 |
|---|---|---|
| ☰ 抽屉按钮 | 28×28 | 左外边距 8 |
| Logo 文字 | FontSize=14, SemiBold | 与抽屉按钮间距 8 |
| 模式 Tab 高度 | 28 | tab 之间间距 4 |
| 模式 Tab 内边距 | 14×4 | — |
| 最近 ▼ 按钮 | 自适应宽度，高度 28 | 与设置按钮间距 4 |
| ⚙ 设置按钮 | 28×28 | — |
| 系统按钮族 | 46×36 | 紧贴右边缘 |

### 2.3 顶栏「最近 ▼」联动规格

| 当前模式 | 按钮文案 | 下拉项数据源 | 下拉项点击行为 |
|---|---|---|---|
| 对话 | 最近会话 ▼ | `SELECT * FROM ChatSessions ORDER BY LastActiveTimestamp DESC LIMIT 10` | 切到该会话 |
| 工作流 | 最近任务 ▼ | `OrchestrationTask WHERE SubMode IN (magentic, parallelanalysis) ORDER BY LastActiveTimestamp DESC LIMIT 10` | 切到该任务详情 |
| 群聊 | 最近群聊 ▼ | `OrchestrationTask WHERE SubMode = groupchat ORDER BY LastActiveTimestamp DESC LIMIT 10` | 切到该群聊详情 |

下拉项样式：每项 `Padding=10,6`，悬停 `Surface0`，最大宽度 300px，超出截断 + ToolTip 显示完整。

| 属性 | 值 |
|---|---|

### 3.4 左侧 Tab2 联动文案表

| 顶部模式 | Tab2 文案 | Tab2 内容 |
|---|---|---|
| 对话 | 会话历史 | `ChatSessions` 列表 |
| 工作流 | 工作流任务 | `OrchestrationTask`（SubMode in magentic/parallel） |
| 群聊 | 群聊任务 | `OrchestrationTask`（SubMode = groupchat） |

**联动时序**：
1. 用户点击顶栏「工作流」tab
2. `MainWindowVm.CurrentMode` 改为 `Workflow`
3. PropertyChanged 触发 → LeftPanel 内：
   - Tab2 文案绑定切换为 "工作流任务"
   - Tab2 内容切换到 `TaskListPanel`
   - 搜索框 PlaceholderText 切换
   - + 新建按钮文案切换为 "+ 新建工作流任务"
   - Tab2 当前激活（不是 Tab1）
4. 同时通知右侧 MainContent 切到 `WorkflowDetailView` 或 `EmptyState`

## 4. 主区（Width = *）

### 4.1 主区三态切换

主区根据 `(CurrentMode, HasContent)` 二元状态切到三类视图：

| 状态 | 视图 | 说明 |
|---|---|---|
| 对话 + 永远 | `ChatView` | 消息列表 + 输入框（对话模式没有空状态） |
| 工作流 + 无任务 | `EmptyState` | 空状态居中视图（决策 UI-5） |
| 工作流 + 有选中任务 | `WorkflowDetailView` | 任务详情（步骤/Token/Report） |
| 工作流 + 列表非空但未选 | `EmptyHintView` | 提示"从左侧选择一个任务"（小提示） |
| 群聊 + 无任务 | `EmptyState` | 同工作流空状态 |
| 群聊 + 有选中任务 | `GroupChatDetailView` | 群聊详情 |
| 群聊 + 列表非空但未选 | `EmptyHintView` | 同工作流 |

### 4.2 主区 ChatView（对话模式）

```
ChatView (DockPanel)
├─ InputArea (Bottom, Height=自适应 80-400)
│  ├─ WorkflowSuggestionBanner (5B Phase 3 已有，按需显示)
│  ├─ AttachmentList (附件预览，水平 WrapPanel)
│  └─ InputBox + SendButton
└─ MessageScroll (Center, *)
   └─ MessageList (消息气泡)
```

### 4.3 主区 EmptyState（空状态）

```
EmptyState (Grid，居中)
└─ StackPanel (HorizontalAlignment=Center, VerticalAlignment=Center, Spacing=16)
   ├─ Image (64×64，模式专属图标)
   ├─ TextBlock (FontSize=18, SemiBold, Foreground=TextBrush, HorizontalAlignment=Center)
   ├─ TextBlock (FontSize=13, Foreground=SubtextBrush, MaxWidth=380, TextWrapping=Wrap, TextAlignment=Center)
   └─ Hyperlink (Foreground=BlueBrush, FontSize=14, Cursor=Hand)
```

完整文案表详见 [02-空状态与文案.md](./02-空状态与文案.md)。

### 4.4 主区 WorkflowDetailView（任务详情）

复用现有 `WorkspaceTab.axaml` 中的详情面板代码（去掉左侧任务列表部分）。详情面板顶部新增任务标题 + 状态徽章 + 取消按钮（已存在）。

## 6. 设计 Token 表

### 6.1 间距（Space）

| Token | 值 | 用途 |
|---|---|---|
| SpaceXs | 4 | 紧凑元素间距、小图标内边距 |
| SpaceSm | 8 | 常规元素间距、按钮 padding 横向 |
| SpaceMd | 12 | 容器内边距、卡片内 spacing |
| SpaceLg | 16 | 区块间距、表单分组 |
| SpaceXl | 24 | 大区块间距、空状态垂直留白 |
| Space2Xl | 32 | 页面 padding、模态对话框 padding |

### 6.2 圆角（Radius）

| Token | 值 | 用途 |
|---|---|---|
| RadiusSm | 3 | 输入框、tab、徽章 |
| RadiusMd | 4 | 按钮、卡片、列表项 |
| RadiusLg | 6 | 弹层、popup、对话框 |
| RadiusXl | 12 | 大型容器、Toast |
| RadiusPill | 999 | 胶囊按钮、状态徽章 |

### 6.3 字号（Font Size）

| Token | 值 | 用途 |
|---|---|---|
| FontXs | 11 | 时间戳、辅助元数据、徽章 |
| FontSm | 12 | 列表说明、次要文案、tab 项 |
| FontBody | 13 | 正文、按钮文字、输入框 |
| FontMd | 14 | 强调正文、Logo |
| FontLg | 16 | 卡片标题、详情标题 |
| FontH2 | 18 | 空状态标题、设置页标题 |
| FontH1 | 22 | 模态对话框标题（如有） |

### 6.4 字重（Font Weight）

| Token | 值 | 用途 |
|---|---|---|
| FontWeightRegular | 400 | 正文、说明、列表项 |
| FontWeightMedium | 500 | 强调正文（按需） |
| FontWeightSemiBold | 600 | 标题、按钮、tab 激活态 |

### 6.5 色彩 Token（沿用 `ThemeResources.axaml`）

完整定义见 `Styles/ThemeResources.axaml`。本设计新增以下别名：

| 别名 | 映射到 | 语义 |
|---|---|---|
| AccentBrush | `BlueBrush #007acc` | 主色 / 链接 / 激活态 |
| AccentHoverBrush | `TealBrush #3794ff` | 主色 hover |
| AccentPressedBrush | 新增 `#005a99` | 主色 pressed |
| SuccessBrush | 新增 `#0e7c3c` | 成功提示 |
| WarningBrush | 新增 `#e0c074` | 警告提示 |
| DangerBrush | `RedBrush #f14c4c` | 危险操作 |
| InfoBrush | `TealBrush #3794ff` | 信息提示 |

### 6.6 阴影（BoxShadow）

| Token | 值 | 用途 |
|---|---|---|
| ShadowSm | `0 2 4 0 #44000000` | 卡片悬浮 |

## 8. 响应式策略

由于决策 UI-10「保持 560 × 980」，本期**不引入响应式断点**。用户拖大窗口后：
- 左侧面板保持当前宽度（不扩张）
- 主区按比例扩张
- 列表项内文字不换行（已有截断处理）

未来若做宽屏适配，新增断点：
- `< 720px`：当前布局
- `720-960px`：左侧默认 280px，主区主区域内嵌侧边详情
- `> 960px`：左 + 主 + 右三栏

详细推到阶段 8+。

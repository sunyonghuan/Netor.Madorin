## 主程序职责

| 组件 | 职责 |
|---|---|
| `VoiceCapabilityRegistry` | 从已加载插件中发现 `voice.kws` / `voice.stt` / `voice.tts` 能力，维护当前选中的插件 |
| `VoiceOrchestrator` | 编排唤醒→问候→识别→AI→合成→续杯流程 |
| `KwsPluginAdapter` | 封装 KWS 插件调用，接收唤醒事件并转发为 `Events.OnWakeWordDetected` |
| `SttPluginAdapter` | 封装 STT 插件调用，接收 partial/final/stopped 并转发为 EventHub 事件 |
| `TtsPluginAdapter` | 封装 TTS 插件调用，接收 started/subtitle/completed 并转发为 EventHub 事件 |
| `SttPluginInputChannel` | 实现 `IAiInputChannel`，收到 STT final 后调用 `IAiChatEngine.SendMessageAsync` |
| `TtsPluginOutputChannel` | 实现 `IAiOutputChannel`，AI token 断句后调用 TTS 插件 enqueue/finish |
| `VoiceCoordinator` | 用户取消时统一停止 KWS/STT/TTS 和当前 AI 任务 |

主程序不再直接引用具体语音模型，不再直接创建 STT/TTS/KWS 引擎实例。

## 语音插件职责

### KWS 插件

| 项 | 职责 |
|---|---|
| 麦克风监听 | 独立进程内长期监听麦克风 |
| 唤醒词检测 | 使用插件自带模型检测关键词 |
| 事件上报 | 检测成功后通过宿主事件通道发送 `voice.kws.detected` |
| 模型管理 | 从插件目录 `models/KWS/` 加载模型 |

### STT 插件

| 项 | 职责 |
|---|---|
| 录音 | 独立进程内录制用户语音 |
| 语音识别 | 输出 partial/final 识别结果 |
| 事件上报 | 发送 `voice.stt.partial` / `voice.stt.final` / `voice.stt.stopped` |
| 模型管理 | 从插件目录 `models/STT/` 加载模型 |

### TTS 插件

| 项 | 职责 |
|---|---|
| 文本合成 | 接收文本并合成为音频 |
| 音频播放 | 独立进程内播放音频 |
| 字幕同步 | 播放时发送 `voice.tts.subtitle` |
| 问候语缓存 | 插件内部缓存欢迎语音频 |
| 模型管理 | 从插件目录 `models/TTS/` 加载模型 |

## 插件能力声明

优先在插件元数据中增加 `capabilities`。如果短期不改 schema，可先使用 `tags` 识别。

```json
{
  "id": "madorin.voice.stt.sherpa",
  "name": "Sherpa STT 语音识别",
  "version": "1.0.0",
  "runtime": "process",
  "command": "Madorin.Voice.Stt.Sherpa.exe",
  "tags": ["voice", "stt"],
  "capabilities": ["voice.stt"]
}
```

能力 ID 约定：

| 能力 | capability |
|---|---|
| 关键词唤醒 | `voice.kws` |
| 语音转文字 | `voice.stt` |
| 文字转语音 | `voice.tts` |

## 编排流程

```text
KWS 插件检测到唤醒词
  │
  ├─ 插件上报 voice.kws.detected
  ├─ KwsPluginAdapter → EventHub: OnWakeWordDetected
  ├─ VoiceOrchestrator 收到唤醒事件
  ├─ 调用 STT 插件 cancel，取消旧识别
  ├─ 调用 TTS 插件 greeting_play，播放问候语
  ├─ 等待 TTS 插件上报 voice.tts.completed
  ├─ 调用 STT 插件 start，开始识别
  ├─ STT 插件上报 voice.stt.partial → EventHub: OnSttPartial
  ├─ STT 插件上报 voice.stt.final → IAiChatEngine.SendMessageAsync
  ├─ AI token → TtsPluginOutputChannel → TTS 插件 enqueue
  ├─ AI done → TTS 插件 finish
  ├─ TTS 插件上报 voice.tts.subtitle → EventHub: OnTtsSubtitle
  ├─ TTS 插件上报 voice.tts.completed → EventHub: OnTtsCompleted
  └─ VoiceOrchestrator 判断是否续杯或结束本轮
```

## 用户主动取消

```text
MainWindow 调 IVoiceCoordinator.CancelEverything()
  │
  ├─ VoiceCoordinator → STT 插件 cancel
  ├─ VoiceCoordinator → TTS 插件 stop
  ├─ VoiceCoordinator → IAiChatEngine.CancelCurrentTask()
  └─ VoiceOrchestrator 重置当前轮次状态
```

## 迁移后的项目方向

| 项目/目录 | 方向 |
|---|---|
| `Src/Netor.Cortana.Voice` | 保留编排、适配器、接口注册，不再内置语音模型引擎 |
| `Src/Netor.Cortana.Voice.Audio` | 如确需复用音频 P/Invoke，可作为插件和主程序共享库；最终主程序可不引用 |
| `Plugins/Src/Cortana.Plugins.Voice.Kws.Sherpa` | 新增 KWS 插件 |
| `Plugins/Src/Cortana.Plugins.Voice.Stt.Sherpa` | 新增 STT 插件 |
| `Plugins/Src/Cortana.Plugins.Voice.Tts.Melo` | 新增 TTS 插件 |
| 主程序发布配置 | 移除内置 `sherpa_models`，模型改随插件发布 |

# 02. 账号身份与用户管理中心

## 1. 账号与身份系统

基础能力：

- 注册、登录、退出。
- 邮箱、手机号、第三方账号绑定。
- 密码登录、验证码登录、OAuth 登录。
- 多因素认证。
- 个人资料、头像、昵称、区域、语言偏好。
- 实名认证和开发者认证。
- 企业组织认证。

建议账号体系支持以下身份状态：

| 状态 | 说明 |
|---|---|
| 未认证用户 | 只能使用基础市场能力 |
| 已认证个人用户 | 可购买、评价、申请开发者 |
| 开发者用户 | 可上传和售卖内容 |
| 企业管理员 | 可管理企业空间和团队订阅 |
| 平台员工 | 可进入管理后台 |
| 超级管理员 | 拥有最高系统配置权限 |

## 2. 用户管理中心

用户中心面向普通用户和企业用户。

### 2.1 我的概览

- 当前订阅数。
- 即将到期订阅。
- 已安装内容。
- 最近购买。
- 推荐升级。
- 安全提醒。

### 2.2 我的订阅

- 订阅列表。
- 状态：有效、试用中、即将到期、已过期、已取消、退款中。
- 续费、取消、升级、降级。
- 查看授权设备。
- 查看使用限制。
- 下载发票。

### 2.3 我的插件 / 技能 / 智能体

- 已购买内容。
- 已安装内容。
- 可更新内容。
- 历史版本。
- 安装到客户端。
- 禁用或卸载。
- 提交反馈。

### 2.4 订单与支付

- 订单列表。
- 支付状态。
- 退款状态。
- 优惠券。
- 支付方式管理。
- 发票申请。

### 2.5 安全与设备

- 登录设备。
- 授权设备。
- API Token。
- 第三方绑定。
- 登录日志。
- 异常提醒。

### 2.6 企业空间

- 成员管理。
- 角色管理。
- 席位分配。
- 团队订阅。
- 企业账单。
- 审计日志。
- 私有内容库。

# 04. 订阅收费与授权体系

## 1. 收费对象

平台可对以下对象收费：

- 单个插件。
- 单个技能。
- 单个智能体。
- 解决方案包。
- 作者会员。
- 企业席位。
- 高级市场服务。
- 官方能力套餐。

## 2. 订阅周期

建议支持：

- 免费。
- 试用 7 天 / 14 天 / 30 天。
- 月付。
- 季付。
- 年付。
- 企业合同制。
- 限时促销订阅。

## 3. 定价模型

| 模型 | 适用场景 |
|---|---|
| 免费 | 基础插件、开源技能、引流内容 |
| 免费试用后订阅 | 高价值插件、智能体、解决方案 |
| 按席位订阅 | 企业团队使用 |
| 按调用量计费 | 云端智能体、API 型能力 |
| 按设备授权 | 本地插件或专业工具 |
| 组合包订阅 | 解决方案套装 |
| 增值服务收费 | 部署、培训、定制、远程支持 |

## 4. 分成模式

建议初始分成策略：

- 平台抽成：20% - 30%。
- 创作者收益：70% - 80%。
- 官方内容：平台 100%。
- 渠道推广：可预留 5% - 15% 佣金池。
- 企业定制：按合同单独配置分成。

结算规则：

- 按月结算。
- 设置 7 - 30 天冻结期，用于退款和风控。
- 达到最低提现金额后可提现。
- 支持结算单、发票、税务扣除和平台服务费明细。

## 5. 授权控制

授权校验需要覆盖：

- 用户是否已购买。
- 订阅是否有效。
- 是否超出设备数。
- 是否超出企业席位。
- 是否允许离线使用。
- 是否允许商用。
- 是否允许二次分发。
- 是否在退款或风控冻结状态。

客户端需要支持本地授权缓存，避免短时离线导致不可用。

## 6. 订阅状态建议

| 状态 | 说明 |
|---|---|
| Trialing | 试用中 |
| Active | 生效中 |
| PastDue | 扣款失败或待补缴 |
| Canceled | 已取消 |
| Expired | 已过期 |
| Refunding | 退款中 |
| Refunded | 已退款 |
| Suspended | 风控冻结 |

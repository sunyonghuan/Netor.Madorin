# 03. 市场内容与资产管理

## 1. 市场内容中心

内容中心用于展示和检索平台商品。

主要功能：

- 首页推荐。
- 分类导航。
- 搜索与筛选。
- 排行榜。
- 新品专区。
- 官方精选。
- 免费专区。
- 企业解决方案专区。
- 插件、技能、智能体、解决方案详情页。

内容详情页建议包含：

- 名称、图标、封面、简介。
- 类型：插件 / 技能 / 智能体 / 解决方案。
- 作者信息、认证标识、历史作品。
- 价格、订阅周期、试用政策。
- 版本号、更新时间、兼容版本。
- 权限声明、运行环境、依赖项。
- 安装量、评分、评论、收藏量。
- 更新日志、使用说明、隐私说明、风险说明。
- 售后支持方式和服务等级。

## 2. 插件管理

插件是具备代码执行能力的高风险资产，需要重点治理。

插件包应包含：

- 插件清单文件。
- 插件二进制或脚本文件。
- 权限声明。
- 依赖声明。
- 入口点声明。
- 版本信息。
- 作者签名。
- 安装前说明。
- 卸载清理说明。

插件管理能力：

- 上传插件包。
- 自动结构校验。
- 自动安全扫描。
- 人工审核。
- 版本发布。
- 灰度发布。
- 紧急下架。
- 兼容性标记。
- 客户端安装授权校验。
- 插件崩溃和异常上报。

插件需要区分以下通道：

| 通道 | 描述 | 风险等级 |
|---|---|---|
| Native DLL 插件 | 直接加载本地 DLL | 高 |
| Process EXE 插件 | 独立进程通信 | 中高 |
| MCP 插件 | 通过 MCP 协议接入工具 | 中 |
| 脚本插件 | PowerShell / Python / JS 等 | 高 |
| 官方内置插件 | 平台官方维护 | 可控 |

## 3. 技能管理

技能通常由提示词、说明文档、工具约束、示例和工作流组成。

技能包建议包含：

- `SKILL.md`。
- 元数据文件。
- 适用场景说明。
- 触发关键词。
- 工具权限要求。
- 输入输出规范。
- 示例任务。
- 安全边界。
- 版本说明。

技能管理能力：

- 创建技能草稿。
- 在线编辑技能说明。
- 导入技能包。
- 自动格式校验。
- 敏感提示词检测。
- 工具调用范围校验。
- 上架审核。
- 版本回滚。
- 客户端同步。

## 4. 智能体管理

智能体是平台中更高层级的能力单元，可以组合技能、插件和模型策略。

智能体配置建议包含：

- 名称、角色、描述。
- 系统提示词。
- 可用技能列表。
- 可用插件列表。
- 可用工具权限。
- 模型偏好。
- 记忆策略。
- 任务流程。
- 安全约束。
- 适用人群和场景。

智能体管理能力：

- 创建智能体模板。
- 上传智能体配置包。
- 调试智能体。
- 发布试用版本。
- 绑定付费订阅。
- 统计调用量、失败率、用户评分。
- 处理滥用、越权和违规输出投诉。

## 5. 解决方案管理

解决方案是面向业务场景的组合商品，例如：

- 办公自动化解决方案。
- 财务报表生成解决方案。
- 企业知识库问答解决方案。
- 桌面自动化助手解决方案。
- 代码开发助手解决方案。
- 客服工单处理解决方案。

解决方案包建议包含：

- 方案说明。
- 包含的插件、技能、智能体清单。
- 依赖关系。
- 安装向导。
- 初始化配置模板。
- 授权策略。
- 售后服务说明。
- 演示示例。

解决方案管理能力：

- 组合多个资产。
- 校验依赖和授权。
- 统一定价。
- 支持一次性试用。
- 支持企业定向授权。
- 支持托管部署和远程协助服务扩展。

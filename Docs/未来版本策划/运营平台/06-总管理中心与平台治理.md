# 06. 总管理中心与平台治理

## 1. 仪表盘

- 平台 GMV。
- 月经常性收入 MRR。
- 年经常性收入 ARR。
- 新增用户。
- 付费转化率。
- 订阅续费率。
- 退款率。
- 待审核内容。
- 风险订单。
- 创作者增长。

## 2. 用户管理

- 用户列表。
- 用户详情。
- 身份认证状态。
- 登录记录。
- 购买记录。
- 订阅记录。
- 举报记录。
- 风控状态。
- 禁用、解封、标记风险。

## 3. 内容管理

- 插件管理。
- 技能管理。
- 智能体管理。
- 解决方案管理。
- 分类、标签、专题、推荐位。
- 评论与评分管理。
- 举报与投诉处理。

## 4. 审核管理

审核对象：

- 插件包。
- 技能包。
- 智能体配置。
- 解决方案组合。
- 作者资质。
- 评论与反馈。

审核结论：

- 通过。
- 驳回。
- 退回修改。
- 限制分发。
- 仅企业可见。
- 冻结版本。
- 下架。

## 5. 订阅管理

- 套餐配置。
- 价格配置。
- 免费试用配置。
- 续费规则。
- 优惠券。
- 订阅状态变更。
- 企业合同订阅。
- 订阅异常处理。

## 6. 支付与财务管理

- 订单流水。
- 支付渠道。
- 退款审核。
- 对账。
- 发票。
- 创作者结算。
- 提现审核。
- 税务数据。
- 财务报表。

## 7. 风控管理

- 异常登录。
- 高频退款。
- 虚假交易。
- 刷评分。
- 恶意插件上传。
- 版权投诉。
- 敏感内容。
- 盗版分发。
- API 滥用。

## 8. 系统配置

- 平台协议。
- 分成比例。
- 审核规则。
- 分类标签。
- 公告配置。
- 版本兼容策略。
- 客户端最低版本。
- 支付参数。
- 邮件短信模板。

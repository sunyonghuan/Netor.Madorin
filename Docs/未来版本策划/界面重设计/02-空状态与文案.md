## 1. 设计原则

### 1.1 三大原则
1. **轻量友好**：使用超链接而非按钮，视觉重量低（决策 UI-5）
2. **明确引导**：标题说"是什么状态"，说明告诉"能做什么"，链接给"怎么做"
3. **品牌一致**：贴合"小娜宝贝"软萌定位，文案温暖不刻板

### 1.2 三段结构

每个空状态都遵循以下三段式：

```
[图标 64×64]
[标题 18px SemiBold #cccccc]
[说明 13px #858585，多行文本，最大宽度 380px]
[链接 14px #007acc，可点击]
[次链接 13px #858585，可选]
```

### 3.3 对话模式

对话模式默认进入就是空白输入框 + 上方滚动消息区。**不展示完整空状态卡片**，而是用更轻量的提示：

| 场景 | 表现 | 说明 |
|---|---|---|
| 全新会话（无消息） | 主区中央展示一行欢迎语 | 嗨，我是小娜宝贝。在下方输入消息开始对话。 |
| 加载历史失败 | Toast 提示 + 顶部 banner | 历史消息加载失败，可以继续新对话或刷新。 |

欢迎语样式：

| 属性 | 值 |
|---|---|
| FontSize | 14 |
| Foreground | `SubtextBrush #858585` |
| HorizontalAlignment | Center |
| VerticalAlignment | Center |
| FontStyle | Italic |
| Opacity | 0.7 |

输入框获得焦点 / 用户开始输入时，欢迎语淡出（200ms）。

**临时降级方案**：图标资产未到位前，先用 Emoji 字符顶替：
- 工作流：📋

- 选择提示：👈

- 加载失败：⚠️
- 文件夹：📁

## 5. 边缘场景与降级文案

### 5.1 加载中状态

不要在加载完成前显示空状态卡片。加载过程使用骨架屏或 spinner：

```
LoadingView
└─ Grid 居中
   └─ StackPanel Spacing=12
      ├─ Spinner 32×32（旋转动画）
      └─ TextBlock "加载中..." (FontSize=12 SubtextBrush)
```

加载阈值：
- < 200ms：什么都不显示（避免闪烁）
- 200-500ms：显示 Spinner
- > 500ms：Spinner + 文案

加载完成后才决定显示空状态 / 列表 / 详情。

### 5.2 离线状态

| 场景 | 文案 |
|---|---|
| 无网络 + 调用 LLM | Toast"网络已断开，无法连接 AI 服务"+ 输入框禁用 |
| 无网络 + 浏览本地数据 | 不影响（本地 SQLite 不依赖网络），不展示提示 |

### 5.3 权限拒绝

| 场景 | 文案 |
|---|---|
| 选择文件夹被拒绝 | Toast"未授予访问权限" |
| 调用工具被任务级黑名单阻止 | 详情面板显示"工具已被本任务屏蔽（pluginId）" |

## 6. 控件规格（EmptyState UserControl）

### 6.1 EmptyState.axaml 设计

```xml
<UserControl x:Class="Netor.Madorin.UI.Controls.EmptyState">
  <Grid>
    <StackPanel HorizontalAlignment="Center"
                VerticalAlignment="Center"
                Spacing="16"
                MaxWidth="380">
      <!-- 图标 -->
      <Image Source="{Binding IconSource}"
             Width="64" Height="64"
             HorizontalAlignment="Center" />

      <!-- 标题 -->
      <TextBlock Text="{Binding Title}"
                 FontSize="18" FontWeight="SemiBold"
                 Foreground="{StaticResource TextBrush}"
                 HorizontalAlignment="Center"
                 TextAlignment="Center" />

      <!-- 说明 -->
      <TextBlock Text="{Binding Description}"
                 FontSize="13"
                 Foreground="{StaticResource SubtextBrush}"
                 HorizontalAlignment="Center"
                 TextAlignment="Center"
                 TextWrapping="Wrap"
                 LineHeight="20" />

      <!-- 主链接 -->
      <Button Classes="btn-link"
              Content="{Binding PrimaryActionText}"
              IsVisible="{Binding HasPrimaryAction}"
              Click="OnPrimaryActionClick"
              HorizontalAlignment="Center" />

      <!-- 次链接 -->
      <Button Classes="btn-link-secondary"
              Content="{Binding SecondaryActionText}"
              IsVisible="{Binding HasSecondaryAction}"
              Click="OnSecondaryActionClick"
              HorizontalAlignment="Center" />
    </StackPanel>
  </Grid>
</UserControl>
```

### 6.2 EmptyState 属性 API

| 属性 | 类型 | 默认 | 说明 |
|---|---|---|---|
| IconSource | string | null | 图标资源路径 |
| Title | string | "" | 标题文本 |
| Description | string | "" | 说明文本（支持多行） |
| PrimaryActionText | string | "" | 主链接文案，空字符串时不显示 |
| SecondaryActionText | string | "" | 次链接文案，空字符串时不显示 |

### 6.3 EmptyState 事件 API

| 事件 | 参数 | 触发时机 |
|---|---|---|
| PrimaryActionClick | RoutedEventArgs | 主链接点击 |
| SecondaryActionClick | RoutedEventArgs | 次链接点击 |

### 6.4 调用示例

```xml
<!-- 工作流模式空状态 -->
<controls:EmptyState
    IconSource="avares://Madorin/Assets/EmptyState/workflow-empty.png"
    Title="还没有任何工作流任务"
    Description="把复杂目标交给多个智能体协作完成。&#10;工作流适合需要分步骤、跨 Agent 协作的任务。"
    PrimaryActionText="✨ 新建工作流任务"
    PrimaryActionClick="OnNewWorkflowTaskClick" />
```

## 1. 验收范围

本阶段验收覆盖以下内容：

1. 宿主侧长期记忆上下文 Provider 是否接入主智能体上下文构建链路。
2. 宿主侧供应客户端是否通过内部 WebSocket/feed 控制面发送 `memory.context.supply.request`。
3. 插件侧是否可接收 `memory.context.supply.request`，并复用 `IMemorySupplyService` 返回结构化 package 或 error。
4. 默认注入是否不再依赖 AI 主动调用 `memory_supply_context` 工具。
5. 插件未连接、超时或错误时是否静默降级，不阻塞主对话。
6. Memory 插件 win-x64 AOT 发布路径是否可通过。

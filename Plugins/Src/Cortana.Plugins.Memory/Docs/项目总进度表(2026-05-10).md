## 风险与处理

| 风险 | 当前处理 |
| --- | --- |
| 插件未启动或未连接 | 宿主返回空 `AIContext`，不影响主对话。 |
| 插件响应慢 | 宿主短超时降级，默认约 250ms。 |
| 记忆污染 prompt | 使用数量、预算、最低可信度和去重过滤。 |
| 作用域错误 | 控制面强制携带 `agentId`，插件缺失即报错。 |
| 插件 AOT 依赖宿主程序集 | 已改为插件本地协议镜像 DTO。 |

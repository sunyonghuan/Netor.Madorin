### M3 AOT 风险记录
- `DesktopPet.Models.Live2D` 使用官方 `Live2DCubismCore.dll` + `LibraryImport`，未使用 Live2D C# NuGet 封装。
- `.model3.json` 使用 System.Text.Json source generation 解析。
- Native AOT 发布成功，Live2D 层未新增 IL warning。
- 当前剩余发布警告仍为 M2 渲染层 Vortice/SharpGen 的 `IL2104`。
- 当前 shader 先通过 `d3dcompiler_47.dll` 运行时编译，便于快速验证；产品化阶段应改成预编译 shader bytecode 并嵌入资源，减少启动依赖与运行时失败点。
- 当前 PNG 解码器为首版限定实现，只支持非交错 8-bit RGB/RGBA PNG，已覆盖 Haru 样例贴图；后续模型管理需要对不支持的 PNG 类型给出明确错误或接入更完整解码方案。
- 当前 clipping mask 为屏幕空间 stencil 首版：每个 masked drawable 绘制前按引用 mask drawable 写入 stencil，再绘制本体。它比官方 mask atlas 慢，但实现清晰、AOT 友好、能先覆盖基础遮罩语义；后续应优化为 mask atlas / render texture 方案。
- 嘴部动作已改为状态入口：`Idle` 回落、`Thinking` 轻微动作、`Speaking` 明显动作；当前 `--preview-speak` 可用于预览，后续 WebSocket / MCP / lipsync 可复用同一状态入口。
